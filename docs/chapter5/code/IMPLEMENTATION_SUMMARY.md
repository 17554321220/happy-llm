## 核心算法

### 权重计算流程
1. **损失输入**: 样本损失 → 权重网络
2. **权重预测**: 神经网络 → 重要性评分
3. **范围约束**: Sigmoid → [0,1] 权重
4. **动量更新**: 历史权重 × 动量 + 新权重 × (1-动量)
5. **加权损失**: Σ(损失 × 权重) / Σ(权重)

### 自适应机制
```python
def compute_weighted_loss(loss_values, weights):
    """
    核心加权损失计算
    高质量样本 → 高权重 → 更大影响
    噪声样本 → 低权重 → 更小影响
    """
    weighted_losses = loss_values * weights
    normalized_loss = torch.sum(weighted_losses) / torch.sum(weights)
    return normalized_loss
```
